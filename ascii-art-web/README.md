# ASCII-Art Web

Ascii-art-web is a graphical user interface (GUI) for the ASCII Art generator, served via a local HTTP server written in Go.

## Description
This project launches a web server that allows users to input text, select a banner style (Standard, Shadow, Thinkertoy), and view the generated ASCII art representation directly in the browser.
